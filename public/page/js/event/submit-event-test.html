submit event test

HTML 입력란에서 엔터로 submit이 발동되는 조건은?

전제 조건: form 태그는 언제나 있을 것

✅ input 하나만 있는 경우

❌ input이 둘 이상인 경우

✅ input이 둘 이상이지만 input[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 button[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 숨겨진 input[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 숨겨진 button[type="submit"]이 있는 경우

⚠️ input은 하나, textarea 하나인 경우

input에서만 엔터로 발동된다.

⚠️ input은 하나, textarea 하나, button[type="button"]이 있는 경우

위와 같음

❌ input은 둘, textarea 하나

결론

form 태그에 input 태그 하나만 있으면 엔터로 submit 이벤트를 발동시킬 수 있다. input이 둘 이상으로 늘어나면 엔터로 발동할 수 없다.

단, input 태그가 여러개여도 submit 버튼(input 혹은 button의 submit 타입 태그)이 존재하면 엔터로 submit 이벤트를 발동시킬 수 있다.

이때 submit 버튼은 보이지 않는 상태(display:none)여도 상관 없음.
